# 发动机舱装配质量智能检测系统
> 实现目标：
  ① 发动舱内微小线划痕检测；
  ② 发动舱内异物检测；
  ③ 发动舱内线距离参数检测。


![](readme.png)

## 使用范例


Windows:

```sh
python win_main.py
```
